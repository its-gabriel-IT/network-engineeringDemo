SOHO Network Connectivity lab
-----------------------------------------------------------------------------------------------------------------------------
Overview

This project simulates a small office home office environment (SOHO) built in Cisco Packet Tracer. It demonstrates fundamental networking concepts.

-----------------------------------------------------------------------------------------------------------------------------

Objectives

*Establish LAN Routing 
*Establish LAN Switching
*Implement Wireless Connectivity
*Implement static IP Addressing

------------------------------------------------------------------------------------------------------------------------------

Technologies & Concepts Used 

*Router Configuration
*Static TCP/IP setting assignment
*Implementing Switching in LAN
*Implementing IEEE 802.11 technology
*Implementing IEEE 802.3 Ethernet Standards
-------------------------------------------------------------------------------------------------------------------------------

Topology Summary:
* 1 Router
* 1 Managed Switch
* Wireless Access Point (WAP)
* Multiple end devices (PCs & Laptops)
* Ethernet RJ45 Cables
-------------------------------------------------------------------------------------------------------------------------------

Skills Demonstrated

* Implementing Basic Networking concepts in a SOHO environment
* Implementing 802.11 Technology in a SOHO environment

-------------------------------------------------------------------------------------------------------------------------------

Files Included

* Packet Tracer file (.pkt)
* Network topology diagram
